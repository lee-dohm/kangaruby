# Kangaruby

A Ruby port of KangaModeling, a text-to-graphics converter for UML-like visualizations.

## Installation

Add this line to your application's Gemfile:

    gem 'kangaruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kangaruby
