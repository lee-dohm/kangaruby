# KangaRuby

A Ruby port of KangaModeling, a text-to-graphics converter for UML-like visualizations.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kangaruby'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install kangaruby
```
